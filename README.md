# DJ Event Booking Website

Eine moderne Webseite zur Buchung von DJ-Services für verschiedene Veranstaltungen.

## Features

✨ **Moderne und responsive Benutzeroberfläche**
- Mobile-freundliches Design
- Smooth Scrolling Navigation
- Gradient-Effekte und Animationen

📋 **Veranstaltungsübersicht**
- 6 verschiedene Veranstaltungstypen
- Pricing-Information
- Detaillierte Beschreibungen

📝 **Buchungsformular**
- Umfassendes Kontaktformular mit:
  - Persönliche Daten (Name, Email, Telefon)
  - Veranstaltungsdetails (Typ, Datum, Uhrzeit, Ort)
  - Anzahl der Gäste
  - Dauer der Veranstaltung
  - Budgetangabe
  - Spezielle Wünsche
- Formularvalidierung
- Datumsvalidierung (nur zukünftige Daten)
- Erfolgsmeldung nach Absendung

📍 **Kontaktinformation**
- Adresse
- Telefonnummer
- E-Mail-Adresse

## Technologie Stack

- **HTML5**: Strukturelle Grundlage
- **CSS3**: Styling mit Gradients und Animationen
- **JavaScript (Vanilla)**: Interaktivität und Formularverarbeitung

## Installation & Verwendung

1. Repository klonen oder Dateien herunterladen
2. `index.html` im Browser öffnen
3. Webseite ist sofort einsatzbereit

## Dateien

- `index.html` - HTML-Struktur der Webseite
- `styles.css` - Styling und Layout
- `script.js` - JavaScript für Interaktivität
- `README.md` - Diese Datei
